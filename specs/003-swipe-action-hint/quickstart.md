## 動作検証

### ヒント表示（US1）

```
1. アプリを起動し、ニュースカードが表示されることを確認する
2. カードをゆっくり右方向にドラッグ → カード左上に緑の「共有」バッジが出ることを確認する
3. カードをゆっくり左方向にドラッグ → カード右上に赤の「興味なし」バッジが出ることを確認する
4. カードをゆっくり上方向にドラッグ → カード上端中央にグレーの「スキップ」バッジが出ることを確認する
5. 閾値未満で指を離す → バッジが消えカードが元の位置に戻ることを確認する
6. 斜め方向にドラッグ → 優勢な方向のバッジのみ表示されることを確認する
```

### 不透明度変化（US2）

```
1. カードをゆっくり右にドラッグしていく
2. 少し動かした段階（5〜30pt）→ バッジが薄く表示されることを確認する
3. さらに引っ張る（hThreshold の 50% 以上）→ バッジが完全に不透明になることを確認する
4. 閾値を超えたままフライアウト → バッジがカードとともに消えることを確認する
```

### 既存動作の非回帰確認

```
1. 右スワイプ完了 → 共有シートが起動することを確認する（ヒントは関係しない）
2. 左スワイプ完了 → 興味なしが記録されカードが消えることを確認する
3. 上スワイプ完了 → スキップされカードが消えることを確認する
4. Undo ボタン → 従来通り動作することを確認する
```
