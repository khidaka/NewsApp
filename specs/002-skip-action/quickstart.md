## 動作検証

### スキップ操作（US1）

```
1. アプリを起動し、ニュースカードが表示されることを確認する
2. カードを上スワイプ → カードが消え、次のカードが表示されることを確認する
3. または、カード右上のスキップボタンをタップ → 同様にカードが消えることを確認する
4. アプリを再起動（Xcode でアプリを停止 → 再起動）
5. ニュースをリロード → スキップした記事が再びスタックに現れる候補となっていることを確認する
   （50件上限の範囲内であれば再表示される）
```

### スキップと「興味なし」の分離確認

```
1. 同じトピックの記事を複数回スキップする
2. パーソナライズスコアが変化していないことを確認する
   → 同トピックの別記事が「興味なし」操作後と比べて表示頻度が落ちていないことで検証
```

### Undo 確認

```
1. 記事をスキップする
2. 画面上部の Undo ボタンをタップ → スキップした記事がスタック先頭に戻ることを確認する
3. 右スワイプ（共有）後に Undo → 記事が戻り共有済みフラグが消えることを確認する
4. 左スワイプ（興味なし）後に Undo → 記事が戻り興味なしフラグが消えることを確認する
```

### 同セッション内のリロード確認

```
1. 記事をスキップする
2. リロードボタンをタップ → スキップした記事がスタックに戻らないことを確認する
```
